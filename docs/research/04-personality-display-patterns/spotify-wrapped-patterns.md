# Spotify Wrapped / Blend — Personality/Compatibility Display Extraction

*Source: Spotify Wrapped screenshots/documentation + Spotify Blend feature + published case studies*

## Personality Data Visualisation (Wrapped)

- **How personal data is shown:** Full-screen vertical story cards, one stat per slide. Each slide is a designed visual — not a data table. Large typography (60px+) for the headline number. Vivid, branded background colours (rotating through Spotify's neon palette: coral, lime, electric blue, hot pink).
- **Colour coding:** Background changes with each slide. Number is the primary element — huge, centred. Supporting label is secondary (smaller, below the number). No data tables, no bars, no axes.
- **Animation:** Each slide "reveals" via a quick zoom-in animation (scale 0.8 → 1.0, 400ms). Numbers count up from 0. Genre labels fly in from the side.
- **Mobile vs desktop:** Vertical swipeable story format on mobile. Web version mirrors the same experience. Optimised for screenshot sharing.

## How Wrapped Feels Celebratory, Not Clinical

**The "Wrapped" formula:**
1. Make the number HUGE — it's the hero, not supporting info
2. Give it a warm, personal frame: "You listened to 14,892 minutes of music this year. That's 10 days straight."
3. Contrast with a relatable anchor: "That's more than 93% of listeners" (social proof without naming others)
4. End with an identity statement: "Your top genre: Indie Pop. You're a trendsetter."

**The progression from data to identity:**
- Data: "14,892 minutes"
- Insight: "10 days straight"
- Social proof: "top 7% of listeners"
- Identity: "You're an Indie Pop fan"

This exact framework should inform how Harmonia presents personality insights:
- Data: "Your Wrath score: -3.8"
- Insight: "You're highly conflict-avoidant"
- Social proof: "Rarer than 87% of Harmonia participants"
- Identity: "The Peacemaker"

## Spotify Blend — Compatibility Between Two Users

Blend creates a shared playlist mixing two users' tastes, then displays a "Taste Match" percentage:

**How Taste Match is displayed:**
- Large percentage number: "62% Taste Match" in the centre of a shared visual
- The shared visual: two overlapping circles (Venn diagram style) in each user's signature colour
- Left circle: Person A's dominant genre, avatar, colour
- Right circle: Person B's dominant genre, avatar, colour
- Overlap zone: "You both love..." — the intersection
- A "shared song" recommendation emerges from the overlap

**Why this is relevant for Harmonia's perceived similarity display:**
- Two overlapping circles (Venn) is a natural metaphor for personality similarity
- Each person has a "colour identity" (comes from their personality type colour-coding)
- The intersection is celebrated, not the divergence
- "You both love..." format = "You're both..." format in Harmonia

## Information Hierarchy

- **What's shown first:** The single most impressive or resonant stat (not the most accurate, the most shareable)
- **Progressive disclosure:** Swipe to see more stats — each is self-contained
- **How much fits on a card:** One stat per slide — the golden rule is ONE KEY IDEA per screen
- **Sharing design:** Every slide is designed for Instagram Stories dimensions (9:16). Screenshot-ready by default.

## Relevant Patterns for Harmonia

**Best approach for making compatibility feel celebratory:**
- Single dominant insight per reveal moment: "You're both more conflict-avoidant than 90% of users"
- Use the Spotify framework: Data → Insight → Social Context → Identity Label
- Design each compatibility reveal as a shareable moment (even if not actually shared)
- Use Harmonia's gold palette where Spotify uses vivid neons — still bold, just warmer

**Best approach for "You're both..." trait cards:**
- One trait per card, swiped horizontally (story-style reveal on the tournament card)
- OR: Show 2-3 traits as a vertical list of short-sentence cards on the reveal panel
- Typography: Trait description in large (20px) serif for warmth, supporting detail in small (13px) sans-serif

**Radar chart on tournament card:**
- Wrapped doesn't use radar charts — they use single-stat screens for simplicity
- This confirms: radar chart is too complex for the tournament card reveal
- Reserve radar chart for the Insights page where users have space to explore
- On the tournament card: use "You're both..." trait pills or a single colour badge

**Phase 3 genetics integration:**
- Spotify's Venn diagram overlap metaphor could work for HLA display: a third ring (biological chemistry) overlapping with the two personality circles
- Label: "Strong chemistry signal" inside the overlap zone
- Or: a secondary badge beneath the personality reveal: "🧬 Strong chemistry" in a green/teal accent

## Visual Notes

- **Typography for data display:** Extra-bold display type (80px+) for the hero number. Cormorant Garamond is a natural Harmonia equivalent for the emotional impact.
- **Colour usage:** Each "moment" has its own vivid background. Harmonia equivalent: gold for match moments, maroon for personality moments, cream for neutral moments.
- **Animation timing:** Slide transitions ~300ms. Number count-up animation 500-800ms.
- **Sharing optimisation:** Design the similarity reveal panel to look good as a screenshot — centred layout, bold typography, Harmonia branding in corner.
